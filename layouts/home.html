Principles

Git-native

All results flow through normal Git review. History, branching, collaboration — the tools you already trust.

Detached by default

Runs happen in isolated workspaces. Your working tree stays untouched until you decide to promote.

Evidence-first

Every run produces durable artifacts — diffs, logs, manifests — that you can inspect, debug, and automate against.

Safe by default

Sandboxed execution with proxy-controlled egress. Unsafe behavior requires explicit opt-in.

What a run leaves behind
.tessariq/runs/01JQ…/
├── manifest.json      # run metadata, adapter, egress config
├── status.json        # lifecycle state, timing, exit code
├── workspace.json     # workspace mode, base SHA, reproducibility
├── task.md            # the original task, versioned
├── diff.patch         # what changed
├── diffstat.txt       # change statistics
├── run.log            # full agent output
└── runner.log         # container lifecycle

Every artifact is JSON, Markdown, or plain text. No proprietary formats. No database required.
